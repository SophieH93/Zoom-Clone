# Zoom Clone

## Add the below to the terminal to get started:
npm init   
press enter   
npm install express   
npm install -g nodemon (-g will install globally)   
to reun the server type 'nodemon server.js' in terminal   
To view site in browser type 'localhost:3030'   

npm install ejs (embedded javascript-Helps get variables from the backend to the frontend)   
npm install uuid (unique id's. This will generate random id's)   

npm install socket.io   
npm install peer   



## Deploy yo Heroku:   
npm install -g heroku    
heroku create   
git push heroku master       
heroku ps:scale web=1   
heroku open   

## Technologies Used:
* [Node.js](https://nodejs.org/en/)
* [Expressjs](https://expressjs.com/)
* [Peerjs](https://peerjs.com/) Automatically generate id's.   
* [socket.io](https://socket.io/)   
* [Heroku](https://www.heroku.com/)
